Important FAQs

What is the Document Object Model (DOM)?

The DOM represents the structure of a webpage in terms of nodes, with attributes and contents that we can access and manipulate using JavaScript.

How do you select an element from the DOM?

We can use either getElementById() or querySelector() to select a single element from the DOM.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation uses the concept of event propagation to allow an event to be triggered by any of the child nodes of a single parent without having to add event listeners to each child element individually, but rather adding one event listener to the parent element.

How do you manipulate an element's attributes and styles using the DOM?

We can use getAttribute(), setAttribute(), and removeAttribute() to manipulate an element's attributes, and classList.add(), classList.remove() to manipulate its style by adding CSS classes to the element.
